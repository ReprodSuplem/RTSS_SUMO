    class Sav::SavDemandFactoryUserAgent

    class for Factory of SavDemand controlled by UserAgent. The config (demandConfig) param should be in the following format:

    <Config> ::= {
               :type => "userAgent",
               :name => <Name>,
               :demand => <MixtureConf>
             }
<Name> ::= name_of_user
<MixtureConf> ::= (see SavDemandFactoryMixture clas)

    (see SavDemandFactoryMixture class.)

    class Sav::TentativeArray

    tentative array for planning. If, the #base array is [a, b, c, d], the #indexList is [2, 3], and objextList is [x, y], then the x and y are inserted just before 2nd and 3rd items so that the resulting array is [a, b, x, c, y, d]. If the index is 0, insert top. If the index is negative, it counts from the last. If the index is -1, it means to insert the last.

def kthIndex(k)
  if(@indexList[k] < 0) then
    return @base.size + 1 + @indexList[k] + k ;
  else
    return @indexList[k] + k ;
  end

def nth(n)
  (0...@indexList.size).each{|i|
    m = kthIndex(i) ;
    if(n < m) then
      return @base[n - i] ;
    elsif(n == m) then
      return @objectList[i] ;
    end
  }
  return @base[n - @indexList.size] ;
end
